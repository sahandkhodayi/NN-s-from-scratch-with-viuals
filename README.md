# 🧠 Neural Networks From Scratch with Visuals

A deep dive into neural networks built entirely from scratch using NumPy, with a strong focus on visualization, mathematical intuition, and understanding how learning actually happens.

This project is part of my journey toward mastering Machine Learning and Artificial Intelligence by implementing the core concepts myself rather than relying on high-level frameworks.

---

## 🎯 Goals

This repository aims to answer questions such as:

* What is a neuron?
* How do weights and biases affect predictions?
* What does a forward pass actually do?
* How does backpropagation work?
* How does gradient descent update parameters?
* How do neural networks learn patterns from data?

Instead of treating neural networks as a black box, every component is built and visualized from the ground up.

## 🎓 Why This Project Exists

Many people learn neural networks by importing a library and training a model.

This repository takes the opposite approach.

The objective is to understand every mathematical operation involved in learning, from the first multiplication to the final prediction.

By the end of this project, every major component of a feed-forward neural network will be implemented manually and explained visually.
